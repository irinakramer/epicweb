# Epicweb notes app

This project is a full-stack notes app that is the result of working through the [Epicweb](https://www.epicweb.dev/)
workshops by [@kentcdodds](https://github.com/kentcdodds):

- Full Stack Foundations

- Professional Web Forms

- Data Modeling Deep Dive

- Authentication Strategies & Implementation

- Web Application Testing

To run the app: `npm run dev`
